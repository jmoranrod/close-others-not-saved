# Close Others Not Saved

Simple VS Code extension to close saved (non-dirty) editors.

## Features

- **Close Saved Files**: Closes all editors that have no unsaved changes across all tab groups.
- **Close Others Saved**: Closes other saved editors in the same group as the selected tab.

## Usage

- Run command `Close Saved Files` from the Command Palette.
- Right-click on an editor tab and select `Close Others Saved`.

## Requirements

VS Code 1.80.0 or higher.
